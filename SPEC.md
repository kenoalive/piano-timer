# 乐器练习记录APP - 项目规范文档

## 一、项目概述

- **项目名称**: 琴时 (Practice Timer)
- **项目类型**: Flutter 移动应用 (iOS/Android)
- **核心功能**: 记录练琴时长、上传练习视频、云端自动同步、统计分析
- **目标用户**: 乐器练习者

## 二、技术选型

### 框架与语言
- **框架**: Flutter 3.4+
- **语言**: Dart
- **最低支持**: Android 5.0+ / iOS 12+

### 核心依赖
| 功能 | 技术方案 |
|------|----------|
| 状态管理 | flutter_bloc (BLoC模式) |
| 本地数据库 | sqflite + shared_preferences |
| 云端存储 | 腾讯云 CloudBase (flutter_cloudbase) |
| 云数据库 | 腾讯云 CloudBase 云数据库 |
| 视频播放 | video_player + chewie |
| 视频录制 | camera |
| 图片/视频选择 | image_picker |
| 图表 | fl_chart |
| 网络状态 | connectivity_plus |
| 时间处理 | intl |

## 四、数据模型

### 4.1 练习记录 (PracticeRecord)
```dart
{
  id: String,              // UUID
  startTime: DateTime,     // 开始时间
  endTime: DateTime?,      // 结束时间 (null=进行中)
  duration: int,           // 时长(秒)
  notes: String?,          // 笔记
  videos: List<Video>,     // 视频列表 (最多9个)
  isSynced: bool,          // 是否已同步到云端
  createdAt: DateTime,
  updatedAt: DateTime
}
```

### 4.2 视频 (Video)
```dart
{
  id: String,              // UUID
  localPath: String,        // 本地路径
  cloudPath: String?,      // 云端路径
  thumbnailPath: String?,  // 缩略图路径
  duration: int,           // 时长(秒)
  size: int,               // 文件大小(字节)
  isSynced: bool,          // 是否已上传
  createdAt: DateTime
}
```

### 4.3 用户设置 (UserSettings)
```dart
{
  dailyGoalMinutes: int,   // 每日目标(分钟), 默认120
  themeMode: ThemeMode,    // 主题模式
  autoSync: bool,          // 自动同步开关
  lastSyncTime: DateTime?, // 上次同步时间
  cloudEnvId: String        // CloudBase环境ID
}
```

## 五、功能模块

### 5.1 计时器模块
- 状态: idle / running / paused
- 最小记录: 1分钟 (不足1分钟不保存)
- 后台运行: 使用 isolates 保持计时
- 数据持久化: 每30秒保存一次进度

### 5.2 视频模块
- 最大数量: 9个/每次练习
- 支持格式: mp4, mov
- 最大单个: 500MB
- 缩略图: 自动生成
- 上传: 后台上传, 支持断点续传

### 5.3 云端同步
- 自动同步: 有网络时每5分钟同步一次
- 手动同步: 设置页可立即同步
- 离线支持: 本地优先, 联网后自动上传
- 冲突处理: 以最新修改时间为准

### 5.4 统计模块
- 时间范围: 周 / 月 / 年
- 柱状图: 显示每日练习时长
- 热力图: 显示练习频率
- 连续天数: 自动计算

## 六、页面详细设计

### 6.1 首页 (HomePage)
- 顶部: 问候语 + 网络状态图标
- 今日累计: 大字体显示
- 计时器: 大字体等宽数字
- 按钮: 开始(绿色) → 暂停/完成
- 笔记: 实时编辑
- 目标: 进度条显示
- 最近视频: 横向滚动列表

### 6.2 完成弹窗 (CompleteDialog)
- 标题: 练琴完成 🎉
- 本次时长: 醒目显示
- 笔记: 可编辑
- 视频: 9宫格上传区
- 操作: 相册选择 / 拍摄视频
- 底部: 跳过 / 保存记录

### 6.3 视频列表页 (VideoListPage)
- 筛选: 全部 / 本月 / 上周
- 分组: 按日期
- 视频项: 缩略图 + 时长 + 时间 + 笔记
- 操作: 播放 / 更多菜单

### 6.4 视频播放页 (VideoPlayerPage)
- 播放器: 16:9 全屏
- 进度条: 可拖动
- 详情: 时长 + 日期
- 笔记: 可编辑
- 操作: 保存 / 删除 / 分享

### 6.5 统计页 (StatisticsPage)
- 切换: 周 / 月 / 年
- 累计时长
- 柱状图 (趋势)
- 统计数据 (平均/连续/本月)
- 热力日历

### 6.6 设置页 (SettingsPage)
- 账号: 在线状态 + 环境ID
- 目标: 步进器调整
- 存储: 使用量 + 清理
- 同步: 自动开关 + 立即同步
- 外观: 浅色/深色/系统
- 数据: 导出/导入/清空
